# Batalha Naval em Python

Jogo de Batalha Naval desenvolvido em Python, utilizando matrizes para representar o tabuleiro e lógica de programação para controlar as jogadas.

## Sobre o projeto

Este projeto foi desenvolvido como prática de lógica de programação, estruturas condicionais, laços de repetição, funções e manipulação de matrizes em Python.

O jogo simula uma partida de Batalha Naval no terminal, permitindo que o jogador escolha posições no tabuleiro e tente acertar os navios.

## Tecnologias utilizadas

- Python
- Matrizes
- Estruturas condicionais
- Laços de repetição
- Entrada e saída de dados pelo terminal
